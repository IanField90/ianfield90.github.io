Whether you're new to Android development or have been doing it in Eclipse for years it's important to be up to speed with a quality targeted IDE. I started my Android development on Windows wrestling with path variables, SDK installation and Eclipse plugins.  Thankfully nowadays I'm more experienced and installation is less of an issue, but the use of such tools has changed a lot.

Since starting with Eclipse at University with embedded libraries, I moved on to using IntelliJ and POMs for dependency management.
